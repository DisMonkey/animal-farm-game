# Animal Farm: Legacy of the Revolution

A strategic narrative game based on George Orwell's Animal Farm, where you lead the animal revolution through seasons, decisions, and the inevitable corruption of power.

## Features

### Core Gameplay
- **Strategic Decision System**: Make choices that affect six core metrics (Loyalty, Rations, Security, Corruption, Innovation, Historical Truth)
- **Seasonal Progression**: Game progresses through Spring, Summer, Autumn, Winter cycles across 6 years
- **Epoch System**: Major narrative events tied to specific years (Windmill Debate, Purges, Commandment Revisions, etc.)
- **Five Complex Endings**: Utopia, Cycle of Tyranny, Stagnation, Martyr's Revolt, or Compromised Survival

### Advanced Features
- **Dynamic Character System**: Interact with Napoleon, Snowball, Boxer, Squealer, and other animals
- **Interactive Farm Map**: Travel between locations (Barn, Windmill, Commandments Wall, Fields, Farmhouse)
- **Reactive Sprites**: Character appearances change based on emotions and game state
- **Undertale-style Dialogue**: Text sounds and character-specific audio
- **Blood Stains System**: Visual representation of historical truth degradation

### Technical Features
- **Persistent Game State**: Auto-save system with multiple slots
- **Responsive Design**: Playable on desktop and mobile devices
- **Web Audio API**: Advanced sound system with mixing and effects
- **Modular Architecture**: Clean separation of concerns with dedicated systems
